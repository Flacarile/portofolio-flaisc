Projects

Katalon Web UI Automation – CURA Healthcare

Role: Test designer & automation engineer (individual project)

Tools: Katalon Studio · Web UI · Test Suite · Data-driven

Mini-project automation untuk login dan booking appointment pada situs demo CURA Healthcare. Satu test case dijalankan berkali-kali dengan kombinasi data berbeda (SUCCESS dan FAIL) menggunakan data-driven testing.

  • Mendesain component login modular yang bisa dipanggil ulang dari berbagai test case.

Learning Tracker Automation – Telegram + n8n + Google Sheets

Role: Workflow designer & automation builder

Tools: n8n · Telegram Bot · Google Sheets · Webhook

Sistem sederhana untuk melacak aktivitas belajar harian: bot Telegram mengirim reminder, lalu setiap balasan (reply) otomatis dicatat ke Google Sheets “Hari ini belajar apa?”. Workflow ini membantu mengalihkan kebiasaan bermain game menjadi habit belajar automation.

  • Membuat workflow n8n dengan Webhook untuk menerima pesan dari Telegram.
  • Menyimpan data ke Google Sheets (tanggal, waktu, catatan) dan menambahkan command seperti /list_today.
  • Merancang struktur ID per catatan untuk rencana fitur /edit langsung dari Telegram.

Loft Bed Frame Concept – Space Saving Study & Wardrobe Area

Role: Personal planning & technical estimation

Focus: Perencanaan rangka · Dimensi ruangan · Estimasi material

Konsep rangka tempat tidur tingkat di atas meja belajar dan lemari pakaian, dirancang untuk ruangan sekitar 280×280×300 cm dengan kasur 90×200 cm. Proyek ini menekankan pada perhitungan dimensi, proporsi sketsa, dan estimasi kebutuhan rangka.

  • Mengatur layout kasur, meja, lemari, dan tangga dalam strip 90×280 cm yang hemat ruang.
  • Menghitung tinggi total sekitar 170 cm agar sejajar dengan lemari sebagai penopang tambahan.
  • Mengestimasi kebutuhan rangka hollow dan biaya material sebagai latihan analisis teknis.
